# Lighting Language

Purpose: select believable light from location, time, and emotional state.

| Lighting State | Emotional Usage | Documentary Application |
|---|---|---|
| Window Light | Honesty, calm, vulnerability | Founder desk, home, interview, morning work |
| Golden Hour | Hope, opportunity, achievement | Payoff, new beginning, city or founder exterior |
| Blue Hour | Transition, uncertainty, anticipation | Commute, city, pre-decision exterior |
| Soft Key | Trust, clarity, authority | Avatar, interview, leadership explanation |
| Hard Key | Pressure, conflict, scrutiny | High-stakes decision, industrial work |
| Negative Fill | Tension, dimensionality, seriousness | Founder reflection, executive pressure |
| Practical Lamp | Intimacy, real work, warmth | Apartment night, desk, late-night founder |
| Laptop Glow | Concentration, isolation | Night work, research, problem solving; keep subtle |
| Rim Light | Separation, confidence, polish | Interview or payoff; never overpower practical light |
| Motivated Light | Believability | Light must appear to come from window, lamp, fixture, or screen |
| Office Fluorescent | Routine, fatigue, corporate pressure | Operational problem, late office, bureaucracy |
| Industrial | Scale, labor, consequence | Factory, warehouse, construction, server room |
| Coffee Shop | Informality, thought, early-stage work | Founder meeting, planning, uncertainty |
| Apartment Night | Solitude, ambition, sacrifice | Solo founder, late-night build, reflection |
| Startup Office | Momentum, collaboration, imperfection | Team work, whiteboards, launch pressure |
| Garage | Experimentation, origin, persistence | Builder story, prototype, early company |
| Warehouse | Scale, logistics, physical consequence | Fulfillment, supply chain, operations |

## Lighting Rules

- Use motivated light in every master image.
- Keep practical sources visible when useful.
- Avoid evenly lit spaces with no direction.
- Use negative fill for tension rather than darkening the entire scene.
- Preserve skin tone realism.
- Do not let laptop glow become sci-fi blue light.
- Shift lighting only when location, time, or emotional state changes.
