# Visual Rhythm

Purpose: prevent repetitive AI-generated sequences and create intentional cinematic progression.

## Script Authority

Apply rhythm through lens, movement, composition, lighting, and edit choices inside the preserved script structure. Never add, remove, merge, split, replace, reorder, or retime shots to satisfy rhythm guidance. Explicit script shot direction always wins.

## Non-Repetition Rule

Never repeat the same lens, same movement, same framing, same distance, or same composition more than twice in succession.

## Automatic Alternation

Use this as the default visual scale cycle:

```text
Wide -> Medium -> Close -> Macro/Detail -> Reaction -> B-roll -> Wide
```

The cycle may be interrupted for story logic, but the next shot must restore variation.

## Rhythm Variables

Track these across the shot list:

- Shot type.
- Lens.
- Camera movement.
- Subject distance.
- Composition.
- Lighting state.
- Color state.
- Human action.
- Information density.

## Emotional Rhythm

- Curiosity: partial reveal, medium, detail.
- Pressure: close, static, negative space.
- Analysis: OTS, insert, neutral medium.
- Momentum: wide, tracking, hand detail.
- Confidence: stable medium close, clean composition.
- Payoff: wide or medium with warm depth and slower pace.

## Block-Level Progression

Visual rhythm must show chapter progression across Blocks 1-5 without changing script structure:

- Block 1: uncertainty, sparse workspace, restrained movement.
- Block 2: thinking, planning details, more deliberate framing.
- Block 3: building, connections, purposeful hand actions.
- Block 4: operations, coordinated movement and active workflow.
- Block 5: momentum, organized depth, stable confidence, visual payoff.

Use only elements established by the current or earlier script blocks. Never preview future maturity.

## Sequence Rules

- Begin a new location with context unless deliberate disorientation is part of the story.
- Follow important information with a reaction or consequence.
- Break montage sequences with a human anchor.
- Avoid three consecutive screen-focused shots.
- Avoid three consecutive moving-camera shots.
- Avoid three consecutive close shots.
- Use meaningful contrast between block endings and beginnings.

## Rhythm QA

Reject the sequence if:

- The same lens appears three times in a row.
- The same movement appears three times in a row.
- Every shot is medium or close.
- Captions create the only sense of change.
- The sequence lacks a human reaction.
- The payoff is cut at the same speed as the problem.
